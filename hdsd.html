Hướng dẫn sử dụng

1. Giới thiệu chung

Chào mừng bạn đến với công cụ tiện ích mở rộng dành cho trình duyệt web, được thiết kế để tối ưu hóa trải nghiệm của bạn trên một số trang trực tuyến.

  • Vượt CORS cho trang tài chính: Hỗ trợ vượt qua lỗi CORS khi bạn truy cập trang drphe.github.io/BM/.
  • Đăng nhập VIP trên VIEON.vn: Cung cấp phương pháp đăng nhập nhanh chóng và tiện lợi vào tài khoản VIP của bạn trên nền tảng giải trí Vieon.vn.
  • Tải File Google PDF only view Cho phép bạn tải xuống các file PDF chỉ cho xem mà không cho tải xuống.

2. Các tính năng chính

Để sử dụng extension hiệu quả nhất, bạn hãy làm theo hướng dẫn dưới đây cho từng tính năng:

Tính năng Vượt CORS

Bật mặc định tính năng vượt CORS cho trang drphe.github.io/BM/.

Đăng nhập VIP trên VIEON.vn

Truy cập Vieon.vn để sử dụng.

Tải file Google PDF only view

Công cụ sẽ hiển thị nút download ở góc bên phải, phía dưới màn hình.
Lưu ý: Nếu file có thể download bình thường thì không nên sử dụng tính năng này.
Cần cuộn xuống cuối trang, để trình duyệt tải hết các trang tài liệu rồi mới nhấn download để tải xuống.
Tài liệu tải xuống là dạng PDF ghép nhiều ảnh, nên không thể copy chữ (text) nếu dùng tính năng này.
